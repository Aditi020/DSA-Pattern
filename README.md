## DSA Patterns - Array

| Pattern | Key Idea | Example Problems |
|---------|---------|----------------|
| Sliding Window | Maintain a window of size K for sums or substrings | Maximum sum subarray of size K, Longest substring without repeating characters, Count of anagrams |
| Two Pointers | Use two pointers to manipulate indices | Pair with a given sum, Merge two sorted arrays, Trapping Rain Water |
| Kadane's Algorithm | Find maximum sum subarray | Maximum Sum Subarray, Maximum Circular Subarray Sum |
| Sorting-based | Sort array and use properties | Triplets summing to zero, Minimum platforms required, Largest number formed from array |
| Binary Search | Efficient search in sorted/rotated arrays | Search in rotated sorted array, Find first/last position of element, Median of two sorted arrays |
| Prefix Sum / Difference Array | Cumulative sum optimization | Subarray sum equals K, Range addition, Equilibrium index |
| Greedy | Local optimal choice for global solution | Maximum meetings in one room, Minimum platforms required, Candy distribution |
| Matrix Variations | 2D pathfinding & traversal | Spiral matrix, Rotate matrix, Search in 2D matrix |
| Hashing | Frequency and presence optimization | Two-sum problem, Longest consecutive subsequence, Count distinct elements in a window |
| Advanced Techniques | Complex patterns for top-tier questions | Maximum product subarray, Subarray with XOR K, Partition array into disjoint intervals |

## DSA Patterns - Strings

| Pattern | Key Idea | Example Problems |
|---------|---------|----------------|
| Sliding Window | Maintain a window to check substrings/frequencies | Longest Substring Without Repeating Characters, Minimum Window Substring, Longest Substring with At Most K Distinct Characters |
| Two Pointers | Manipulate string indices | Valid Palindrome, Reverse Words in a String, Longest Palindromic Substring, Compare Backspaced Strings |
| Prefix Sum / Cumulative Frequency | Use prefix sums or hashmaps | Substring with equal 0s and 1s, Longest Substring with Balanced Parentheses |
| Hashing (Frequency Count) | Store character frequencies | Group Anagrams, First Unique Character, Longest Substring with K Unique Characters |
| Dynamic Programming | Break into subproblems | Longest Palindromic Subsequence, Edit Distance, Longest Common Subsequence, Wildcard Matching |
| Backtracking | Explore all possibilities | Generate All Permutations, Word Break Problem, Palindrome Partitioning |
| String Searching Algorithms | Efficient substring/pattern search | KMP Algorithm, Rabin-Karp Algorithm, Z-Algorithm |
| Trie | Prefix searching | Longest Word in Dictionary, Search Suggestions, Autocomplete |
| Bit Manipulation | Use bitmasking for unique chars | Maximum Length of Concatenated String, Subsets with Unique Characters |
| Regular Expressions | Pattern matching | Regex Matching, Wildcard Matching |
| Mathematical String Problems | Numerical string operations | Add Binary Strings, Multiply Strings, Decode Ways |
| Greedy Algorithms | Optimal local choices | Rearrange Characters to Avoid Repetition, Minimum Swaps to Make Strings Equal |
| Compression & Encoding | Encoding/compressing strings | Run-Length Encoding, Count and Say, Decode String |
| Miscellaneous | Rotations, shifts, conversions | Check if two strings are rotations, Minimum shifts, String to Integer Conversion, Repeated substrings |

## Stack, Queue & Linked List

- **Stack**
  - Monotonic Stack/Queue: Next Greater Element, Largest Rectangle, Sliding Window Maximum
  - Stack Operations: Valid Parentheses, Reverse Polish Notation
- **Queue**
  - Implement LRU Cache, Circular Queue
- **Linked List**
  - Classic: Reverse, Detect & Remove Cycle, Merge Two Sorted Lists
  - Advanced: Flatten Multilevel Doubly Linked List, Add Two Numbers
